# kiro-test

A test repository for the Upslope Pivot agentic dev workflow POC.

This project is used to validate and demonstrate the Kiro agentic developer workflow, including automated ticket-driven development, code generation, and repository management.

## Tech Stack

- [Next.js](https://nextjs.org/) — React framework for production
- [TypeScript](https://www.typescriptlang.org/) — Typed JavaScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
